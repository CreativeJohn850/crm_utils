## 3. Lists

### Unordered List

* Item 1
* Item 2
    * Sub-item 2.1
    * Sub-item 2.2
* Item 3

### Ordered List

1. First item
2. Second item
3. Third item
    1. A sub-list item
    2. Another sub-list item

### Task List

- [x] Task 1 (completed)
- [ ] Task 2 (not completed)
- [ ] Task 3
